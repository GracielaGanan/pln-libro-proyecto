Proyecto PLN: Limpieza de texto de un libro

Descripción

En este proyecto trabajé con un libro en formato `.txt` para aplicar técnicas básicas de procesamiento de lenguaje natural. La idea fue tomar texto real y limpiarlo para dejarlo listo para futuros análisis.

 Qué se hizo?

Primero cargué el archivo del libro y luego apliqué distintas etapas de limpieza:

* Pasé todo el texto a minúsculas
* Eliminé signos de puntuación y caracteres innecesarios
* Quité palabras vacías (stopwords)
* Separé el texto en palabras (tokenización)
* Apliqué lematización para llevar las palabras a su forma base

Estructura del proyecto

El repositorio contiene:

* El archivo `.txt` con el libro
* El notebook o script con todo el desarrollo
* El archivo `requirements.txt` con las librerías utilizadas

 Tecnologías usadas

Trabajé con Python y librerías de PLN como NLTK o spaCy, ejecutando el código en Google Colab.

Resultado

Al final obtuve un texto limpio y procesado, listo para usar en análisis más avanzados como conteo de palabras, clasificación o modelos de lenguaje.

 Cómo usarlo

1. Clonar el repositorio
2. Instalar las dependencias con `pip install -r requirements.txt`
3. Abrir el notebook o ejecutar el script
